Your ChatGPT workspace.

ChatGPT Ether is a personal Chrome extension for managing prompts, conversations, timelines, starred messages, and safe metadata sync around the ChatGPT web app.

Core Features

Prompt Vault

Save, organize, search, tag, and insert reusable prompts.

Conversation Manager

Save ChatGPT conversations, organize them with folders, and add notes.

Timeline & Search

Browse the current conversation timeline and search within the open conversation.

Starred Messages

Mark important messages and jump back to their conversation context.

Google Drive Sync

Manually sync safe extension metadata to your own Google Drive appDataFolder.

Diagnostics

Inspect extension status without collecting or uploading diagnostics logs.

Privacy Boundaries

The extension does not

  • Read cookies or browsing history
  • Request all_urls permission
  • Call ChatGPT private APIs
  • Upload or sync full chat transcripts
  • Sell user data

Optional Google Drive sync

  • Uses Google Drive appDataFolder
  • Syncs prompts, folders, notes, settings, and starred-message metadata
  • Does not sync full ChatGPT conversation bodies
  • Runs only after user authorization and manual sync actions
