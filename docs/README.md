# ChatGPT Ether Docs

[简体中文](./README.zh-CN.md)

ChatGPT Ether is a personal Chrome extension for managing ChatGPT conversations and prompts.

Current scope:

- Prompt Vault
- Conversation folders and index
- Timeline and current conversation search
- Starred message metadata
- Google Drive manual sync for safe extension data
- Diagnostics

Privacy boundaries:

- Does not read cookies.
- Does not read browsing history.
- Does not request `all_urls`.
- Does not sync full chat content.
- Does not upload attachments, images, screenshots, or raw conversation JSON.

For project-level attribution, notices, and privacy details, see the root `CREDITS`, `NOTICE`, and `PRIVACY` files.
