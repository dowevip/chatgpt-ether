ChatGPT Ether is a Chrome extension focused on ChatGPT web conversation management.

This project was able to take shape quickly thanks to upstream open-source work and community projects.

## Upstream Project

### Nagi-ovo/gemini-voyager

Repository: [https://github.com/Nagi-ovo/gemini-voyager](https://github.com/Nagi-ovo/gemini-voyager)

ChatGPT Ether is based on Gemini Voyager. Gemini Voyager provided the original extension project structure, browser-extension foundation, and part of the implementation direction.

This project has been heavily refactored, trimmed, and renamed for the ChatGPT use case, but it should continue to clearly attribute the upstream project.

## Inspiration

### Reborn14/chatgpt-conversation-timeline

Repository: [https://github.com/Reborn14/chatgpt-conversation-timeline](https://github.com/Reborn14/chatgpt-conversation-timeline)

ChatGPT Ether's timeline locating and message navigation design was inspired by parts of this project, especially the direction of using ChatGPT page-native message identifiers when possible.
