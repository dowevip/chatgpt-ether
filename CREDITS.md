ChatGPT Ether / ChatGPT以太 is a Chrome extension focused on ChatGPT conversation management.

## Upstream and Inspiration

* [Nagi-ovo/gemini-voyager](https://github.com/Nagi-ovo/gemini-voyager)
  This project is based on / adapted from Gemini Voyager. Gemini Voyager provided the original extension foundation and many inherited implementation patterns.

* [Reborn14/chatgpt-conversation-timeline](https://github.com/Reborn14/chatgpt-conversation-timeline)
  Timeline navigation ideas for ChatGPT conversation pages were inspired by this project.
